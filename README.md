# Maison Alure Frontend

Production-style frontend for Maison Alure built with Next.js, TypeScript and Tailwind CSS.

## Run

```bash
npm install
npm run dev
```

## Main routes

- `/` Home
- `/about`, `/how-it-works`, `/trust`, `/membership`, `/one-time-rentals`
- `/catalog`, `/catalog/[slug]`, `/ai-styling`, `/community`
- `/delivery-returns`, `/contact`, `/faq`, `/partnerships`
- `/signup`, `/login`, `/onboarding`
- `/account/*` account area
- `/states/empty`, `/states/success`, `/states/error`
